WindowsTools for Python

Overview

WindowsPyTools is a Python library dedicated to accessing native Windows features (displays, system, audio, applications, notifications‚Ä¶).

Project status

WindowsPyTools is currently under development. The API may change and some features are still experimental.

The documentation reflects the current state of the project and may evolve alongside the code.

Installation

Compatible with Windows only. 
pip install -i https://test.pypi.org/simple/ WindowsTools-Tipang-test
This version contains only some of the features listed below; the latest version is not yet available for download.

Goals

  • Provide a simple API for Windows features
  • Rely only on the Python standard library
  • Be installable and usable like a regular Python module

Quick start

Retrieve information about connected displays:

from WindowsPyTools import DisplayInfo

display = DisplayInfo()
print(display.get_monitor_count())

Change the system volume:

from WindowsPyTools import SystemManager

SystemManager.Volume.set_volume(50)

DisplayInfo

The DisplayInfo module allows you to retrieve information about displays and monitors connected to the system.

Methods

get_monitor_count()

Returns the number of currently connected displays.

display.get_monitor_count()

get_primary_monitor()

Returns the index of the primary display.

display.get_primary_monitor()

get_all_monitors_info()

Returns a list containing information about each connected monitor.

display.get_all_monitors_info()
The returned information includes position, resolution, screen bounds, and whether the monitor is primary or not.

get_monitor_info(index)

Returns information about a specific display.

display.get_monitor_info(0)
As with get_all_monitors_info(), the returned information includes position, resolution, screen bounds, and primary status.

is_primary(index)

Indicates whether the given display is the primary one.

display.is_primary(0)

refresh()

Reloads the list of connected displays.

display.refresh()

WallpaperManager

The WallpaperManager module allows you to manage the Windows desktop wallpaper, either globally or per monitor.

Features

  • Change the global desktop wallpaper
  • Set a different wallpaper for each monitor (depending on Windows version)

Methods

set_global_wallpaper(path)

Sets a single wallpaper for all connected monitors. 

WallpaperManager.set_global_wallpaper("wallpaper.jpg")

set_wallpaper_per_monitor([(index, path), ...])

Per-monitor wallpaper management may depend on the Windows version and user permissions.

Sets a specific wallpaper for each monitor independently. 

WallpaperManager.set_wallpaper_per_monitor(
    [(0, "wallpaper1.png"), (2, "wallpaper3.jpeg")]
)
