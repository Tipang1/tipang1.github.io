Project name

WindowsPyTools is a Python library dedicated to native Windows features, with no external dependencies.

Goals

  • Clean and simple API
  • Standard library only
  • Classic pip installation
  • Windows-only by design

Quick start

from WindowsPyTools import DisplayInfo

display = DisplayInfo()
print(display.get_monitor_count())
