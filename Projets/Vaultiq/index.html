Vaultiq

Vaultiq is an offline password manager for Windows. All data is stored locally in an encrypted vault, with no cloud, no account, and no internet connection required.

Goals

  • Provide a simple and fast password manager
  • Ensure secure local encrypted storage using a master password
  • Remain fully offline with no online services or accounts
  • Stay lightweight and easy to use on a daily basis
